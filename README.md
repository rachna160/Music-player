A modern web-based music player inspired by Netflix UI, featuring background videos, playlists, favorites, and Google login using Firebase.

---

## Features

- **Music Player**
  - Play, pause, next, previous
  - Song progress and volume control
  - Favorite songs list
  - Add custom songs via file upload

- **Background Videos**
  - Each song can have a looping video
  - Uploaded songs can use default video

- **Dark/Light Mode Toggle**
  - Switch between dark and light themes seamlessly

- **Search Songs**
  - Filter songs by title

- **User Authentication**
  - Login using Google (Firebase Authentication)
  - Displays user photo, name, and email
  - Logout functionality

- **Persistent Storage**
  - Songs list and favorites saved in localStorage
